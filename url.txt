>>> 서버 실행 순서

1. python version: 3.11.6 설치
2. python 시스템 환경변수 path 설정
3. C 드라이브 내 'Digital Twin Simulation Program' 폴더 생성 또는 git clone

    C:\Digital Twin Simulation Program.
    ├─ auto simulation
    ├─ static
    ├─ templates
    ├─ .env
    ├─ .gitignore
    ├─ requirements.txt
    ├─ url.txt
    ├─ windows.py

4. 해당 경로 내 python 가상환경 생성

    (1) cmd 실행
    (2) 해당 디렉터리 이동 명령어 'cd C:/Digital Twin Simulation Program' 실행
    (3) python 가상환경 생성 명령어 'python -m venv venv' 또는 'python3 -m venv venv' 실행
    
        C:\Digital Twin Simulation Program.
        ├─ auto simulation
        ├─ static
        ├─ templates
        ├─ .env
        ├─ .gitignore
        ├─ requirements.txt
        ├─ url.txt
        ├─ windows.py
        ├─ venv          <= 가상환경 생성

5. python 가상환경 실행
    해당 디렉터리에서 명령어 'cd venv/Scripts' 실행
    'C:/Digital Twin Simulation Program/venv/Scripts' 이동 후 명령어 'activate' 실행
    다시 명령어 'cd C:/Digital Twin Simulation Program' 실행 디렉터리로 이동.

6. package 설치
    명령어 'pip install -r requirements.txt' 실행

7. 서버 실행
    명령어 'python app.py' 실행
